Isomorphic [Flux](http://facebook.github.io/flux) application for checking out the meals served by the [University Restaurant](http://www.pcasc.usp.br/ser_restaurante.htm) at [USP SÃ£o Carlos](http://www.saocarlos.usp.br). Built using [React](http://facebook.github.io/react), [Fluxible](http://fluxible.io), [webpack](http://webpack.github.io) and [Express](http://expressjs.com).

## Usage

```bash
$ git clone https://github.com/rbardini/bandeco && cd bandeco
$ npm install && npm start
```

Open up http://localhost:3000 and _bom apetite!_
